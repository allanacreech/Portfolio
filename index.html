Technical Competencies

HTML MARKUP LANGUAGE

HTML is the standardized system including the newly released version of the classification for tagging text files, including graphics, and hyperlink effects on each of my clients World Wide Web pages.

CSS Color and Sizzle

Cascading Style Sheets aids me to help set the mood for each of my clients projects. Making multiple web pages under the same control works with the 'look' of the layout, width, height, or design. If you want sparkle or design this allows me to meet your needs.

jQuery Javascript Library

The jQuery Library allows a concise and fruiously fast Javascript to run when handling simplified events, moving through the HTML, and working towards speedy animations.

NODE Js Javascript Framework

Node.js is a framework that uses Javascript for non-sequential output. It allow me to ask mutliple items to process at one time and to do it before anyone task is complete. Fast-n-Furious.

Heroku Hosting and Database

Heroku is a cloud-based development tool that supports mutliple languages and platforms. It allows me to live test the site to see different functionalities and database management.

GITHUB Version Control System

GitHub is a online hosting service that allows me to keep track of the different versions of the code that I am working on. It is a Source Code Management that works for backing up our information.

RWD Responsive Web Design

Responsive Web Desing allow me to have your desktop web page to be viewed anywhere on any item. It responds to what device is accessing the information so it will look great on a phone or your PC.

ADOBE JPG, PNG, Vector

Adobe creative software enables me to create items from scratch to meet my clients needs. If its removing a backgroung or creating an web app icon lets do this!
